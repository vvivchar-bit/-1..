# Система пропускного контролю в 18 корпус

Навчальний вебпрототип системи пропускного контролю, розроблений для лабораторних робіт з теми виявлення та моделювання вимог до програмного продукту.

## Варіант

**Варіант №7 — Система пропускного контролю в 18 корпус**

## Опис проєкту

Система призначена для автоматизації контролю доступу студентів, викладачів, працівників університету та відвідувачів до 18 корпусу.

Прототип дозволяє:

- перевіряти право доступу користувача;
- додавати нових користувачів;
- змінювати статус доступу;
- фіксувати вхід та вихід;
- переглядати журнал відвідувань;
- експортувати журнал у CSV;
- зберігати дані у браузері через LocalStorage.

## Запуск проєкту

Для запуску потрібно відкрити файл:

[Відкрити index.html](<index.html>)

Або завантажити репозиторій і відкрити файл `index.html` у браузері.

- [style.css](<style.css>) — файл стилів
